# CryptoVest

## Overview
CryptoVest is a cryptocurrency investment tool designed to simplify and enhance the user experience of managing digital assets. With a user-friendly interface and powerful features, CryptoVest allows users to track their investments, analyze market trends, and make informed trading decisions.

## Features
- **Portfolio Management**: Easily manage your cryptocurrency investments in one place.
- **Market Analysis**: Access real-time data and trends to stay updated on market movements.
- **User-Friendly Interface**: Navigate seamlessly through the application with a responsive design.
- **Notifications**: Set alerts for price changes and market news.

## Installation
1. Clone the repository: `git clone https://github.com/fairandjustjames-afk/CryptoVest.git`
2. Navigate to the project directory: `cd CryptoVest`
3. Install the dependencies: `npm install`
4. Start the application: `npm start`

## Usage
After starting the application, you can:
- Create a new portfolio.
- Add or remove cryptocurrencies from your portfolio.
- View market data for specific cryptocurrencies.
